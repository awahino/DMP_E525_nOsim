# Data open repositories for E525 paper and n-O simulation paper.
The ploted data in papers, E525 experiment, and n-O simulation, are released in this repository.
Please follow the description under each subdirectory:
- [Measurement of gamma-rays generated by neutron interaction with 16O at 30 MeV and 250 MeV](e525_ptep/README.md)
- [A simulation model investigation of neutron-oxygen inelastic scattering and subsequent nucleus deexcitation based on experimental data](sim_epjc/README.md)

- Hino, Y., Ashida, Y., Tano, T. et al. A simulation model investigation of neutron-oxygen inelastic scattering and subsequent nucleus deexcitation based on experimental data. Eur. Phys. J. C 86, 148 (2026). [doi](https://doi.org/10.1140/epjc/s10052-026-15355-2)
